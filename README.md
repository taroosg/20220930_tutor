## 課題提出用アプリケーション

- 進捗 10%（設計考え中のみ，10 月前半でなんとかします w）

- FRESH で実装予定

  - [https://fresh.deno.dev/](https://fresh.deno.dev/)

- 実装機能の予定

  - アプリケーションから提出できる（受講生）．

  - スプレッドシート上で課題の評価ができる（チューター）．

  - スプレッドシートの内容を反映してアプリ上で提出内容や評価を確認できる（みんな）．

  - 発表した課題や過去の偉大な作品をアプリ上で確認できる（みんな）．
